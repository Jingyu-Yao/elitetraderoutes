About

EliteTradeRoutes provides an easy way to search for details on various game data related to trading for the game Elite: Dangerous. The first version of this website will be focused on providing a REST API as well as an equally featured web view.

You can find specific game data by using the search box or by browsing the database through the categories listed in the nav bar.

API

This site also provides a REST API. Any data you can see on the web will also be available via the API. Currently no there are no API keys but please go easy on the server. To specify the return type of the content you can either use the appropriate accept header (recommended) or append '.json', '.html' or '.api' to the end of your calls. ex. /systems/1/.json will return the data for that endpoint in json and /systems/1/.api will display api information for that endpoint.

The use of accept header is recommended because if a suffix is used, all of the hyperlinks returned on that page will also have the suffix. This can become somewhat bothersome as it makes the return value not very consistent.
